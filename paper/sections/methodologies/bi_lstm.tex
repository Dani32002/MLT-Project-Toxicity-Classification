\subsection{BI-LSTM}
\subsubsection{Definition}

BI-LSTM (Bidirectional Long Short-Term Memory) is a type of sequential neural network that can work with data 
composed of different elements arranged in a sequence, such as texts. This type of network can make predictions 
about parts of the sequence based on what has already been seen, or about the entire input. It represents an improvement 
over LSTM networks, which, like recurrent networks, use the output values from previous steps in the sequence. However,
it differs in that it divides the network's responsibilities into different control flow gates. Furthermore, it is capable of maintaining both a long-term memory and a hidden state, allowing it to respond appropriately to the current step in the sequence. BI-LSTMs process input in both forward and reverse directions, thus capturing future and past context at each step.
